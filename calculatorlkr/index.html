How it works

Five steps. Completely covert.

01
Open the "Calculator"

Install and open the app. You see a normal, functional calculator. Nothing suspicious — tap some numbers, it all works.

02
Unlock the Vault

Type your secret PIN as if entering a calculation (e.g. 1234), then press =. The vault unlocks silently.

03
Lock Files (Encrypt)

Choose "Lock Files" from the hidden menu. Pick files via the Android picker. They get encrypted into secure private storage.

04
Unlock Files (Decrypt + Export)

Open the vault, pick the file to restore, and export it to any folder you choose using SAF — safe and private.

05
Recover Your PIN

Long-press C on the calculator screen to access the PIN recovery screen. Answer your security question to reset.
